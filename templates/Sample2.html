A tool that uses NLP to get the insights of the data. Here you can select a domain and we will fetch some live data related to that topic and then processing is done using various nlp libaries and other modules. Using one of the sentence from the fetched data, the named-entity relationship is shown,alongwith that one dependency pattern is also shown that provides how words are inter-related in a sentence and also, the sentiment analysis of the entire data is done which is shown as a graph. All of the mentioned images and graphs can be viewed by clicking on the respective buttons on the next page that will redirect the user to the given page. Also,the top entity data and the data after sentiment analysis that provides the sentiment(Positive, Negative , Neutral) alongwith its sentiment score that can be found in the local folder.
